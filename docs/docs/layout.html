布局Layout
该框架通过JSON配置形成文档结构,最外层是PageView组件,可以通过设置PageView的样式来控制整体的布局结构, 比如是垂直三行布局或者是垂直两行布局或者其他。
PageView通过包含View等容器组件来做到局部的布局。可以调整View的样式来控制局部的样式展示。
该框架建议开发者使用Flex语法进行页面布局,本框架也对Flex布局以及低版本的Box布局进行了兼容。
开发这只需要使用Flex语法进行配置声明。
例子中只是列出了行布局的例子,列布局的例子调整flexDirection:"column"
