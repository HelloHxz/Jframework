业务工程目录
这里以Demo项目为例
目录简介
  • css文件夹
    存放项目的样式文件
  • dist文件夹
    打包生成文件存放目录详细
  • fonts文件夹
    业务项目的字体存放目录,因为自带的字体名为icomoon所以添加的字体不能与之重名
  • logic文件夹
    业务项目逻辑文件的存放目录
  • pages文件夹
    业务项目页面UI文件的存放目录
  • app.js文件
    业务项目的启动文件,可以在这里进行全局变量配置,启动页,及全局的一些拦截机制详细
  • index.html文件
    开发时的项目入口,真正的打包后的文件入口在dist中
  • index.tpl文件
    开发时的项目入口打包模版文件,根据该模版自动生成dist中的index.html
  • gulpfile.js文件
    gulp打包逻辑文件
