重复Repeat
该组件用于展示多项重复的组件。功能大体类似于ListView但是该组件用于展示无需分页的数据。
  • 可以实现九宫格,Menu的效果
  • 可以实现单选Menu,多选Menu的组件
  • 可以配合其他容器组件实现多种效果
  • 提供各种数据绑定,数据更新,行操作的API
Repeat组件是RepeatItem组件的集合
UI配置选项
  • typestring
    指定组件的类型为repeat
  • rootarray
    行第一级子组件的Key的集合,用于描述行的字组件的组成
  • stylejson
    列表整体的样式配置
  • itemsArray
    通过该数据制定组件的数据,也可以通过后期动态API绑定
  • nodatastring
    制定列表在没有数据的时候展示的组件Key
  • itemStylejson
    行的样式配置,比如可以指定行的排列方向
Repeat 属性
  • datasourceArray
    数据源
  • bindData(datasource)function
    绑定数据
  • addItem(itemData,index)function
    插入一行数据
RepeatItem 属性
  • typestring
    制定组件的类型为listview
