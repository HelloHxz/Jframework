页面PageView
这是最核心的组件,整个UI文件配置就是页面组件的配置。
基本结构

define(["../logic/逻辑页面名称"],function(pluginClass){
    return {
        pluginClass:pluginClass,
        style:{
            backgroundColor:"#f0f4f6"
        },
        root:[],
        components:{

        }
    };
});
UI属性
  • pluginClassobject
    定制页面后台处理对象实例
  • rootArray
    制定页面的第一级的组件Key数组
  • stylejson
    样式配置
  • componentsjson
    页面所有组件的配置,以key-value的形式存储组件的配置
