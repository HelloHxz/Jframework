导航Navigator
路由的API包含在页面实例对象里。页面的实例对象可以在逻辑对象的构造函数中获得。(详见项目核心-逻辑文件)
1.基本路由
  • go(pageKey,params)function
    @param {string} pageKey 跳转的页面名称
    @param {json} params 页面传递的参数,因为是通过URL传递的参数,所以值只能是字符串
  • goBackfunction
    回退,在目标页面进行调用
  • replaceGo(pageKey,params)function
    跳转页面,清除了当前页面的路由记录
    @param {string} pageKey 跳转的页面名称
    @param {json} params 页面传递的参数,因为是通过URL传递的参数,所以值只能是字符串
  • prePageViewobject
    在目标页面,可以通过目标页面实例对象获取到前一个页面的引用
  • paramsobject
    在目标页面,可以通过目标页面实例对象获取到前一个页面的参数传递
2.弹出页面popPage
  • showPage(options)function
    @param {JSON} options 参数配置
    {
    pageKey:"xxpagename",
    mode:"fromBottom",
    params:{

    }
}
    @param {string} pageKey 目标页面名称
    @param {string} mode 弹出方式 具体有以下几种fromBottom- fromLeft1 fromLeft2- fromLeft3- fromRight1- fromRight2- fromRight3
    @param {json} params 可以传递复杂的JSON对象
  • hideCurShowPagefunction
    关闭页面中弹出的页面,在宿主页面进行调用,效果同等于在目标页面调用goBack
  • goBackfunction
    回退,在目标页面进行调用
  • ownerPageobject
    在目标页面,可以通过目标页面实例对象获取到宿主页面的引用
  • showPageParamsobject
    在目标页面,可以通过目标页面实例对象获取到宿主页面的参数传递
