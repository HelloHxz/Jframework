运行项目
步骤
  • 使用命令行达到工程根目录
  • npm install 这一步会安装Express,以及gulp插件
  • npm start
Mac环境和Windows环境一样
node服务会开启3333端口
打开Chrome浏览器
http://localhost:3333/Apps/demo/index.html
打开浏览器的开发者模式,进行手机模拟,刷新页面
框架默认提供两个现成的项目
1. demo
包括全部控件Demo展示
地址:http://localhost:3333/Apps/demo/index.html
2. starterkit
项目初始化的空项目
地址:http://localhost:3333/Apps/starterkit/index.html
每次新建一个项目,可以复制拷贝整个starterkit在同级目录,比如取名为project
地址:http://localhost:3333/Apps/project/index.html
